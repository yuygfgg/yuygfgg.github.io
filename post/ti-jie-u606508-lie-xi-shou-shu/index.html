题解：U606508 裂隙收束

2025-09-02 OI

1. 问题概述与分析

本问题要求通过重复执行“收束”操作（将相邻且阶数同为 kkk 的裂隙合并为 k+1k+1k+1 阶），最小化最终剩余的裂隙数量。给定序列长度 N≤2×107N \le 2 \times 10^7N≤2×107，要求算法具有接近线性的时间复杂度（如 O(Nlog⁡N)O(N \log N)O(NlogN)）。此外，输出操作序列时必须使用操作发生时的动态索引。

2. 贪心策略的局限性与证明

在求解最优化问题时，贪心算法通过在每一步选择中都采取当前状态下最好或最优的选择，从而希望导致结果是全局最好或最优的。然而，本问题的核心特征在于，任何一次合并操作都是不可撤销的，并且会消耗序列中的元素，从而永久性地改变未来可行的操作集。一个局部的最优选择可能会破坏形成更长、更有效合并链的潜力，因此贪心策略无法保证全局最优。

2.1 贪心策略的定义

一个位置优先贪心策略 可以定义为：在算法的任意一步，如果存在多个可合并的相邻对 (Ai,Ai+1)(A_i, A_{i+1})(Ai​,Ai+1​) 使得 Ai=Ai+1A_i = A_{i+1}Ai​=Ai+1​，该策略会根据一个预设的、确定性的位置规则（如“选择最小的 iii”或“选择最大的 iii”）来选择其中一对进行合并，并持续此过程直至无对可合。

2.2 贪心策略的非最优性

考虑一个序列 S=[k,k,k,k+1]S = [k, k, k, k+1]S=[k,k,k,k+1]，其中 k≥1k \ge 1k≥1 为任意整数。

  1. 贪心策略的执行路径：

    • 该策略从左向右扫描，发现的第一个可合并对是位于索引 0 和 1 的 (k,k)(k, k)(k,k)。
    • 执行合并后，序列变为 S1=[k+1,k,k+1]S_1 = [k+1, k, k+1]S1​=[k+1,k,k+1]。
    • 在此新序列中，不存在任何相邻且相等的元素。因此，合并过程终止。
    • 贪心策略得到的最终序列长度为 3。

  2. 存在更优的解：

    • 在初始序列 S=[k,k,k,k+1]S = [k, k, k, k+1]S=[k,k,k,k+1] 中，存在另一个选择：合并位于索引 1 和 2 的 (k,k)(k, k)(k,k) 对。
    • 执行该合并，序列变为 S1′=[k,k+1,k+1]S'_1 = [k, k+1, k+1]S1′​=[k,k+1,k+1]。
    • 继续扫描，发现新的可合并对 (k+1,k+1)(k+1, k+1)(k+1,k+1)。
    • 执行合并后，序列变为 S2′=[k,k+2]S'_2 = [k, k+2]S2′​=[k,k+2]。
    • 合并过程终止。此路径得到的最终序列长度为 2。

结论： 由于 2<32 < 32<3，存在一条非贪心路径可以得到更优的结果。因此，从左向右的贪心策略不是最优的。其根本错误在于，它优先执行的 A0,A1A_0, A_1A0​,A1​ 合并，破坏了后续 A1,A2A_1, A_2A1​,A2​ 与 A3A_3A3​ 形成更优组合的可能性。

同理可证，从右往左的贪心策略也不是最优的。

3. 动态规划与优化

我们将问题转化为：将原序列分割为最少数量的连续子段，使得每个子段都可以最终收束成一个单一的裂隙。

3.1 状态定义与转移

定义 DP[i]DP[i]DP[i] 为后缀子序列 A[i…N−1]A[i \ldots N-1]A[i…N−1] 能够收束成的最少裂隙数量。

DP[i]=min⁡_j=iN−1{1+DP[j+1]}DP[i] = \min\_{j=i}^{N-1} \{ 1 + DP[j+1] \}DP[i]=min_j=iN−1{1+DP[j+1]}
条件是：子段 A[i…j]A[i \ldots j]A[i…j] 能够收束成单一的裂隙。

优化的关键在于如何高效地找到所有满足条件的 jjj。

3.2 预处理

我们可以观察到一个关键性质：对于固定的起点 iii，如果它能收束成单一裂隙，那么所有可能的结果值必然形成一个从 A[i]A[i]A[i] 开始的连续整数区间，即 {A[i],A[i]+1,A[i]+2,…}\{A[i], A[i]+1, A[i]+2, \ldots\}{A[i],A[i]+1,A[i]+2,…}。

我们利用这个性质进行预处理。目标是对于每个 iii 和每个可达的值 VVV，快速找到对应的结束位置 jjj。

预处理算法:

我们采用逆序遍历（iii 从 N−1N-1N−1 到 000）。当我们处理 iii 时，所有 k>ik>ik>i 的信息已经计算完毕。

  1. 初始化: 子段 A[i…i]A[i \ldots i]A[i…i] 收束为值 V=A[i]V=A[i]V=A[i]，结束位置 j=ij=ij=i。
  2. 迭代扩展: 尝试将当前形成的 VVV（对应子段 A[i…j]A[i \ldots j]A[i…j]）扩展为 V+1V+1V+1。这需要找到一个相邻的“伙伴”子段，起始于 k=j+1k=j+1k=j+1，它也能收束成 VVV。
  3. 快速查找: 我们利用已计算的 kkk 的信息来查找伙伴。由于可达值是连续的，如果 V≥A[k]V \ge A[k]V≥A[k] 且 VVV 在 kkk 的可达范围内，我们可以在 O(1)O(1)O(1) 时间内找到伙伴子段的结束位置 j′j'j′。
  4. 合并与重复: 如果找到伙伴，则 A[i…j′]A[i \ldots j']A[i…j′] 可以收束为 V+1V+1V+1。更新 V←V+1V \leftarrow V+1V←V+1，j←j′j \leftarrow j'j←j′，并继续尝试扩展。

复杂度分析: 该预处理的总时间复杂度取决于总的可达状态数。由于形成一个较大的值 VVV 所需的序列长度通常呈指数级增长，总状态数被有效限制在接近 O(N)O(N)O(N) 的级别。

3.3 执行动态规划

预处理完成后，DP 过程可以直接利用预处理结果高效转移。对于每个 iii，我们遍历其所有可达状态，找到对应的结束位置 jjj，并根据 1+DP[j+1]1 + DP[j+1]1+DP[j+1] 更新 DP[i]DP[i]DP[i]。同时记录最优决策以便重构方案。

4. 方案重构与动态索引管理

本题要求输出操作时的动态位置。由于每次收束都会导致后续索引前移，我们需要高效管理索引映射。

4.1 使用树状数组

我们使用树状数组（BIT）来维护动态索引。初始化一个逻辑上全为 1 的数组，表示所有裂隙均存在。

  • 查询动态位置: 原始索引为 iii 的裂隙的当前动态位置，等于树状数组中前缀 [0…i][0 \ldots i][0…i] 的和（即在它之前还剩多少个裂隙）。
  • 更新索引: 当发生一次收束，一个裂隙消失（例如右侧块起始于原始索引 kkk），我们在树状数组的 kkk 位置执行减 1 操作。这会自动更新后续位置的前缀和，正确反映索引左移。

4.2 递归重构操作序列

我们根据 DP 记录的最优决策划分序列，并对每一段递归重构其内部操作。重构过程遵循后序遍历的逻辑：

  1. 拆分: 要形成目标值 VVV，必须由两个值为 V−1V-1V−1 的子段（左侧 LLL 和右侧 RRR）合并而成。根据预处理记录找到这两个子段的边界。
  2. 递归执行:
    • 首先递归执行 LLL 内部的操作。
    • 然后递归执行 RRR 内部的操作。
  3. 执行当前合并: 此时 LLL 和 RRR 已收束且相邻。查询树状数组以获取 LLL 的当前动态位置并输出。随后，更新树状数组以反映 RRR 的消失。

递归的顺序保证了输出的动态位置是正确的。重构阶段总共有 O(N)O(N)O(N) 次合并操作，每次操作涉及 O(log⁡N)O(\log N)O(logN) 的树状数组操作，因此总时间复杂度为 O(Nlog⁡N)O(N \log N)O(NlogN)。

5. 代码实现

由于输入输出量较大，需要使用快读快输。

#include <cctype>
#include <cstdint>
#include <cstdio>
#include <functional>
#include <vector>
using namespace std;

using ll = int64_t;

struct FT {
    int n;
    vector<int> b;
    inline void init(int _n) {
        n = _n;
        b.assign(n + 1, 0);
        for (int i = 1; i <= n; ++i)
            b[i] = i & -i;
    }
    inline void add(int i, int v) {
        for (; i <= n; i += i & -i)
            b[i] += v;
    }
    inline int sum(int i) const {
        int r = 0;
        for (; i > 0; i -= i & -i)
            r += b[i];
        return r;
    }
};

struct IO {
#define MAXSIZE (1 << 20)
#define isdigit(x) (x >= '0' && x <= '9')
    char buf[MAXSIZE], *p1, *p2;
    char pbuf[MAXSIZE], *pp;

    IO() : p1(buf), p2(buf), pp(pbuf) {}

    ~IO() { fwrite(pbuf, 1, pp - pbuf, stdout); }
    char gc() {
        if (p1 == p2)
            p2 = (p1 = buf) + fread(buf, 1, MAXSIZE, stdin);
        return p1 == p2 ? ' ' : *p1++;
    }

    void read(int& x) {
        bool neg = false;
        x = 0;
        char ch = gc();
        for (; !isdigit(ch); ch = gc())
            if (ch == '-')
                neg = true;
        if (neg)
            for (; isdigit(ch); ch = gc())
                x = x * 10 + ('0' - ch);
        else
            for (; isdigit(ch); ch = gc())
                x = x * 10 + (ch - '0');
    }

    void read(int64_t& x) {
        bool neg = false;
        x = 0;
        char ch = gc();
        for (; !isdigit(ch); ch = gc())
            if (ch == '-')
                neg = true;
        if (neg)
            for (; isdigit(ch); ch = gc())
                x = x * 10 + ('0' - ch);
        else
            for (; isdigit(ch); ch = gc())
                x = x * 10 + (ch - '0');
    }

    void read(char* s) {
        char ch = gc();
        for (; isspace(ch); ch = gc())
            ;
        for (; !isspace(ch); ch = gc())
            *s++ = ch;
        *s = 0;
    }

    void read(char& c) {
        for (c = gc(); isspace(c); c = gc())
            ;
    }

    void push(const char& c) {
        if (pp - pbuf == MAXSIZE)
            fwrite(pbuf, 1, MAXSIZE, stdout), pp = pbuf;
        *pp++ = c;
    }

    void write(int x) {
        bool neg = false;
        if (x < 0) {
            neg = true;
            push('-');
        }
        static int sta[40];
        int top = 0;
        do {
            sta[top++] = x % 10;
            x /= 10;
        } while (x);
        if (neg)
            while (top)
                push('0' - sta[--top]);
        else
            while (top)
                push('0' + sta[--top]);
    }

    void write(int x, char lastChar) { write(x), push(lastChar); }
} io;

int main() {
    int N;
    io.read(N);
    if (N == 0)
        return 0;

    vector<ll> a(N);
    for (int i = 0; i < N; ++i)
        io.read(a[i]);

    if (N == 1) {
        io.write(1, '\n');
        return 0;
    }

    vector<int> st(N), len(N);
    vector<int> pool;
    pool.reserve(N * 2);

    for (int i = N - 1; i >= 0; --i) {
        int s = (int)pool.size();
        int j = i;
        ll v = a[i];
        pool.push_back(j);
        while (true) {
            int k = j + 1;
            if (k >= N)
                break;
            ll pos64 = v - a[k];
            if (pos64 < 0)
                break;
            int pos = (int)pos64;
            if ((unsigned)pos < (unsigned)len[k]) {
                j = pool[st[k] + pos];
                ++v;
                pool.push_back(j);
            } else {
                break;
            }
        }
        st[i] = s;
        len[i] = (int)pool.size() - s;
    }

    const int INF = 0x3f3f3f3f;
    vector<int> dp(N + 1, 0), bt(N, -1), be(N, -1);
    dp[N] = 0;
    for (int i = N - 1; i >= 0; --i) {
        int bestC = INF, bestT = -1, bestJ = -1;
        int base = st[i];
        for (int t = 0; t < len[i]; ++t) {
            int j = pool[base + t];
            int c = 1 + dp[j + 1];
            if (c < bestC || (c == bestC && t > bestT)) {
                bestC = c;
                bestT = t;
                bestJ = j;
            }
        }
        dp[i] = bestC;
        bt[i] = bestT;
        be[i] = bestJ;
    }

    FT fw;
    fw.init(N);
    vector<int> ans;
    ans.reserve(N);

    function<void(int, int)> rec = [&](int i, int t) {
        if (t == 0)
            return;
        int split = pool[st[i] + (t - 1)];
        int k = split + 1;
        int tr = (int)((a[i] + (ll)t - 1) - a[k]);

        rec(i, t - 1);
        rec(k, tr);

        ans.push_back(fw.sum(i + 1));
        fw.add(k + 1, -1);
    };

    for (int i = 0; i < N;) {
        rec(i, bt[i]);
        i = be[i] + 1;
    }

    for (int x : ans)
        io.write(x, '\n');

    io.write(fw.sum(N), '\n');

    return 0;
}
