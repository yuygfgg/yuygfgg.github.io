在s390x架构的ubuntu22.04上安装tailscale

2024-03-21 ubuntu ， tailscale ， s390x

很难想象这种需求，但我确实有
ubuntu的apt不提供tailscale的s390x架构的包，所以要自己编译

安装go
sudo snap install go
用go安装tailscale

go install tailscale.com/cmd/tailscale@latest tailscale.com/cmd tailscaled@latest

运行tailscaled
sudo /home/username/go/bin/tailscaled

运行tailscale

sudo /home/username/go/bin/tailscale up --advertise-exit-node

然后就好啦
